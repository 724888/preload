## 图片预加载介绍

>从字面上来看：预，就是预先；预加载：即预先提前加载。
>
>图片预先加载到浏览器中，访问者可以顺利的网站上浏览冲浪，并享受极快的加载速度；

我们为什么需要预加载？

某些情况下丰富的图片资源，带来更好的用户体验的同时，也降低的我们的性能，有时候，我们在观看页面时需要不断的等待页面加载图片，带来的极差的用户体验(这种表现在画廊，有大量照片、图片占据比例大的地方尤为明显)；
